# Open-Source 3D Printed Hand Exoskeleton

A low-cost wearable robotic hand exoskeleton developed to help stroke survivors regain hand movement, improve grip strength, and support rehabilitation therapy.

## Key Features

- Low-cost and portable design
- 3D printable mechanical system
- Rotary encoder based control
- Assistive gripping mode
- Therapy training mode
- Improves finger range of motion (ROM)

## Repository Contents

- CAD design files
